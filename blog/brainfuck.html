// Brainfuck

Posted on October, 1st 2021

After my blog about Wireworld, I discovered a programming language called 'Brainfuck' and it pretty much does as it says. Some people prefer to call it Brainf***, Brainbleep, Brainf*ck or even Br*infuck but I'll just stick with the original name from now on.

Languages like Brainfuck are called esoteric languages, basically the point of them is that they're funny and completely impractical to use. Malbolge is just one example so just try and guess what this does.

There are only 8 different characters in total: + - [ ] <> , .
Anything else will be ignored when the script is run. Imagine a long line of numbers all start at 0. The < and> characters select the number on the left and right respectively. The + and - characters increase and decrease the number.

The other 4 are a bit harder. The . converts the currently selected number to an ASCII character and prints it to the console. The , gets an inputed character from the user and sets the currently selected number to that character's ASCII code. Lastly, if the current selected value is 0 then the [ character will skip to its corresponding ]. If the value isn't zero at the ], then it will rerun everything in between the []s. This may sound complicated but basically it will loop over the code in the []s until the currently selected number is 0.

Brainfuck is also a Turing-tarpit, meaning it could technically be programmed to complete the Turing test but its completely unrealistic and unreasonable to do so. Writing 'Hello world' is hard enough as it is.
