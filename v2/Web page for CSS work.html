The following languages platy a key role in the creation of a modern web page:

  • HTML
  • CSS
  • JavaScript

CSS (Cascading Style Sheets), like HTML, is a client-side language. In fact, it works very closely with HTML, and its role is to control the visual formatting / style / design of a web page.

The difficulties with web design pre-CSS

Before CSS, it was possible, using HTML, to control the visual appearance of your text, your buttons etc., but there was a huge problem. If your website contained hundreds of pages, and you wanted a fresh visual makeover for the whole website, then you would have to go to the HTML code for each element on every page to change the style.

CSS to the rescue

The principle behind CSS is that you separate the style / design code and put it in a CSS file, whilst the content of your web page remains within an HTML file. It's a bit like putting someone in charge of making a product in your company (HTML), and somebody else in charge of marketing (CSS).

A different HTML page on your website can use the same CSS file to maintain the same visual style. This makes it very easy to change the style of the whole site â€“ you just need to edit the CSS file.
